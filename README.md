## Configuration

In the rcon console:

  gather.rate dispenser * 2
  oxide.grant default furnacesplitter.use
  stackall 2000

In oxide/config/MagicLoot.json

```json
{
  "_Settings": {
    "MultiplyOnlyItemList": true,
    "RandomWorkshopSkins": true
  }
  //...
}
